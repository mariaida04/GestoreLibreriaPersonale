# GestoreLibreriaPersonale
Progetto "Gestione di una libreria personale" relativo al corso di Ingegneria del Software, A.A. 2024/2025.

# Descrizione
Implementazione di un'applicazione in Java per la gestione di una collezione personale di libri. 
All'utente sono consentite le operazioni di aggiunta, modifica, rimozione, ricerca, filtraggio e ordinamento di libri.
